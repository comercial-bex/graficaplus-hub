# Matriz de Permissões

Perfis suportados: administrador, gerente, financeiro, vendedor, designer, operador, estoque, instalador, cliente. O legado `admin` mapeia para administrador e `gestor` para gerente.

O catálogo completo está em `src/lib/permissions.ts` e é espelhado no banco pelas tabelas `permissoes` e `perfil_permissoes`.
